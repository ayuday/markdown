# Markdown 编辑器推荐

选择合适的编辑器能大大提升 Markdown 写作体验。本页面推荐各种类型的 Markdown 编辑器，帮助你找到最适合的工具。

## 专业 Markdown 编辑器

### Typora
> 真正的所见即所得 Markdown 编辑器

**特点**：
- ✅ 实时渲染，无需分屏预览
- ✅ 优雅的用户界面
- ✅ 丰富的主题支持
- ✅ 数学公式、图表支持
- ✅ 导出多种格式（PDF、Word、HTML）

**适用场景**：
- 长文写作
- 学术论文
- 技术文档
- 日常笔记

**价格**：付费软件（约 $14.99）
**平台**：Windows、macOS、Linux

### Mark Text
> 简洁的实时预览编辑器

**特点**：
- ✅ 免费开源
- ✅ 实时预览
- ✅ 专注模式
- ✅ 支持多种格式导出
- ✅ 轻量级

**适用场景**：
- 个人写作
- 技术博客
- README 文件

**价格**：免费
**平台**：Windows、macOS、Linux

### Zettlr
> 学术写作专用编辑器

**特点**：
- ✅ 针对学术写作优化
- ✅ 文献管理集成
- ✅ Zettelkasten 方法支持
- ✅ 强大的搜索功能
- ✅ 多种导出选项

**适用场景**：
- 学术论文
- 研究笔记
- 文献整理

**价格**：免费
**平台**：Windows、macOS、Linux

## 代码编辑器

### Visual Studio Code
> 最受开发者欢迎的编辑器

**推荐插件**：
- **Markdown All in One** - 全功能 Markdown 支持
- **Markdown Preview Enhanced** - 增强预览功能
- **markdownlint** - 语法检查
- **Paste Image** - 快速插入图片

**特点**：
- ✅ 强大的插件生态
- ✅ Git 集成
- ✅ 多文件管理
- ✅ 自定义性极强
- ✅ 智能补全

**配置示例**：
```json
{
  "markdown.preview.fontSize": 16,
  "markdown.preview.lineHeight": 1.6,
  "markdown.extension.toc.levels": "2..6",
  "markdown.extension.preview.autoShowPreviewToSide": true
}
```

**适用场景**：
- 技术文档
- 项目 README
- 代码注释文档
- 多文件项目

**价格**：免费
**平台**：Windows、macOS、Linux

**推荐包**：
- **MarkdownEditing** - 增强 Markdown 编辑

**特点**：
- ✅ 极快的性能
- ✅ 多选和命令面板

- ✅ 插件系统

**适用场景**：

**价格**：付费（$99，可免费试用）

- ✅ 极高的编辑效率

## 选择指南

### 按使用场景选择

| 场景 | 推荐编辑器 | 理由 |
|------|------------|------|
| 学术写作 | Typora, Zettlr | 数学公式、文献管理 |
| 技术文档 | VS Code | 版本控制、插件丰富 |
| 个人笔记 | Obsidian, Notion | 知识管理、链接功能 |
| 快速编辑 | Sublime Text | 性能优秀、启动快 |
| 团队协作 | Notion, HackMD | 实时协作、权限管理 |
| 移动写作 | iA Writer, Markor | 移动优化、同步功能 |

### 按技术水平选择

**初学者**：
- Typora - 所见即所得，易上手
- Notion - 界面友好，功能直观
- Mark Text - 简单纯粹

**进阶用户**：
- VS Code - 功能强大，可定制
- Obsidian - 知识管理，插件丰富
- Sublime Text - 高效编辑

**专业用户**：
- Vim/Neovim - 最高效率
- Zettlr - 学术写作专业化
- 自定义工具链

### 按预算选择

**免费**：
- VS Code
- Mark Text  
- Obsidian（个人）
- Logseq
- Markor

**付费但值得**：
- Typora（$14.99）

## 编辑器配置技巧

### VS Code 优化配置

```json
{
  // Markdown 相关设置
  "markdown.preview.fontSize": 16,
  "markdown.preview.lineHeight": 1.6,
  "markdown.extension.toc.levels": "2..6",
  
  // 编辑器设置
  "editor.wordWrap": "on",
  "editor.lineNumbers": "off",
  "editor.minimap.enabled": false,
  
  // 专注模式
  "zenMode.centerLayout": false,
  "zenMode.hideActivityBar": true,
  "zenMode.hideStatusBar": true
}
```

### Typora 自定义主题

```css
/* 自定义 CSS */
#write {
  max-width: 800px;
  margin: 0 auto;
  line-height: 1.6;
  font-size: 16px;
}

h1, h2, h3, h4, h5, h6 {
  color: #2c3e50;
  margin-top: 2em;
}

blockquote {
  border-left: 4px solid #3498db;
  padding-left: 1em;
  color: #7f8c8d;
}
```
