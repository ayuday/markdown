# Markdown 链接速查表

## Markdown 链接语法

### Markdown 内联链接

```markdown
[链接文本](https://www.markdownlang.com)
[带标题的链接](https://www.markdownlang.com "链接标题")
```

### Markdown 引用式链接

```markdown
[链接文本][ref]
[链接文本][]

[ref]: https://www.markdownlang.com "可选标题"
[链接文本]: https://www.markdownlang.com
```

### Markdown 自动链接

```markdown
<https://www.markdownlang.com>
<email@example.com>
```

### Markdown 锚点链接

```markdown
[跳转到标题](#heading-name)
[返回顶部](#top)
```
