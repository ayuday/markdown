# 标题语法速查

## ATX 风格标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## Setext 风格标题

```markdown
一级标题
========

二级标题
--------
```

## 标题 ID（扩展语法）

```markdown
### 标题 {#custom-id}
### 标题 {.class-name}
```

## 标题链接

```markdown
跳转到 [标题](#标题)
```

## 最佳实践

- ✅ 标题层级要有逻辑性
- ✅ 避免跳跃层级（如 # 直接到 ###）
- ✅ 每个文档只有一个一级标题
- ❌ 不要在标题中使用链接
- ❌ 避免标题过长

## HTML 输出

| Markdown | HTML |
|----------|------|
| `# 标题` | `<h1>标题</h1>` |
| `## 标题` | `<h2>标题</h2>` |
| `### 标题` | `<h3>标题</h3>` |
