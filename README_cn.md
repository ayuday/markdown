## 🛠️ 实用工具集

基于Vue3和UnoCSS构建的现代化Markdown工具集：

### 📝 Markdown 编辑器
实时预览编辑器，支持语法高亮和工具栏操作
- **功能**: 左右分屏预览、语法高亮、快捷工具栏
- **地址**: `/tools/editor`

### 🔄 格式转换工具
- **Markdown转HTML**: 将Markdown转换为HTML代码，支持下载
- **Markdown转PDF**: 将Markdown文档导出为PDF文件
- **HTML转Markdown**: 将HTML代码转换为Markdown格式

### 📖 Markdown 查看器
专门的只读预览工具，支持URL加载和文件上传
- **功能**: 美观预览、文件上传、URL加载
- **地址**: `/tools/viewer`

### 📋 语法速查表
交互式速查表，点击查看示例和效果
- **功能**: 分类展示、实时示例、快速复制
- **地址**: `/tools/cheatsheet`

## 📚 工具集使用指南

### 环境要求

- **Node.js**: >= 16.0.0
- **pnpm**: >= 8.0.0 或 **npm**: >= 7.0.0 或 **yarn**: >= 1.22.0
- **现代浏览器**: Chrome 88+, Firefox 85+, Safari 14+, Edge 88+

### 项目结构

```
src/
├── package.json          # 项目依赖配置
├── vite.config.js        # Vite 构建配置
├── uno.config.js         # UnoCSS 配置
├── index.html            # 入口 HTML
└── src/                  # 源代码目录
    ├── main.js           # Vue 应用入口
    ├── App.vue           # 主应用组件
    ├── router.js         # 路由配置
    ├── components/       # 工具组件
    │   ├── AppHeader.vue     # 导航头部
    │   ├── AppFooter.vue     # 页脚
    │   ├── MarkdownEditor.vue    # Markdown 编辑器
    │   ├── MarkdownViewer.vue    # Markdown 查看器
    │   ├── MarkdownToHtml.vue    # MD 转 HTML
    │   ├── MarkdownToPdf.vue     # MD 转 PDF
    │   ├── HtmlToMarkdown.vue    # HTML 转 MD
    │   └── MarkdownCheatsheet.vue # 语法速查表
    ├── utils/            # 工具函数
    │   └── markdown.js   # Markdown 处理工具
    ├── views/            # 页面组件
    │   └── Home.vue      # 首页
    └── styles/           # 样式文件
        └── main.css      # 全局样式
```

### 工具功能详解

#### 📝 Markdown 编辑器 (`/editor`)

**功能特性：**
- 左右分屏实时预览
- 丰富的工具栏（格式化、标题、列表、链接等）
- 键盘快捷键支持
- 语法高亮显示
- 支持导出 MD 和 HTML

**使用方法：**
1. 在左侧编辑区输入 Markdown 内容
2. 右侧实时显示渲染效果
3. 使用工具栏按钮快速格式化文本
4. 支持键盘快捷键（如 Ctrl+B 粗体）
5. 可下载编辑的内容

#### 📖 Markdown 查看器 (`/viewer`)

**功能特性：**
- 支持 URL 加载 Markdown 文件
- 支持本地文件上传
- 美观的预览样式
- 文档统计信息
- 支持源码和预览模式切换

**使用方法：**
1. 输入 Markdown 文件的 URL，或上传本地文件
2. 自动渲染预览效果
3. 可切换源码/预览模式
4. 支持复制内容和下载

#### 🔄 Markdown 转 HTML (`/to-html`)

**功能特性：**
- 实时 Markdown 到 HTML 转换
- 支持下载完整的 HTML 文件
- 内置预览功能
- 复制 HTML 代码

**使用方法：**
1. 在输入框粘贴或输入 Markdown 内容
2. 右侧自动显示转换后的 HTML
3. 点击"复制"按钮复制 HTML 代码
4. 点击"下载HTML"获取完整的 HTML 文件

#### 📄 Markdown 转 PDF (`/to-pdf`)

**功能特性：**
- Markdown 转 PDF 导出
- 支持多种页面格式（A4、A3、Letter、Legal）
- 自定义边距设置
- 可选择纵向/横向布局
- 可选页眉页脚

**使用方法：**
1. 输入 Markdown 内容
2. 配置 PDF 参数（页面大小、方向、边距等）
3. 点击"生成PDF"下载文件

#### 🔄 HTML 转 Markdown (`/html-to-md`)

**功能特性：**
- HTML 到 Markdown 的智能转换
- 支持复杂的 HTML 结构
- 保持原有格式
- 内置示例模板

**使用方法：**
1. 在输入框粘贴 HTML 代码
2. 右侧显示转换后的 Markdown
3. 选择内置示例快速测试
4. 下载转换结果

#### 📋 Markdown 速查表 (`/cheatsheet`)

**功能特性：**
- 交互式语法参考
- 分类展示所有语法
- 点击查看详细示例
- 支持搜索和过滤

**使用方法：**
1. 浏览不同分类的语法
2. 点击卡片查看详细用法
3. 使用搜索快速查找
4. 复制示例代码

### 技术栈说明

#### 前端框架
- **Vue 3**: 使用 Composition API 和 `<script setup>` 语法
- **Vue Router 4**: 单页应用路由管理

#### 样式和UI
- **UnoCSS**: 原子化 CSS 框架，提供丰富的工具类
- **Tailwind CSS**: 通过 UnoCSS 的预设支持
- **自定义样式**: 项目特定的样式定制

#### Markdown 处理
- **Marked**: 快速的 Markdown 解析器
- **Turndown**: HTML 转 Markdown 转换器
- **Prism.js**: 代码语法高亮（可选集成）

#### 工具库
- **html2pdf.js**: HTML 转 PDF 导出
- **@vueuse/core**: Vue 组合式函数工具集

#### 构建工具
- **Vite**: 快速的开发服务器和构建工具
- **ESLint**: 代码质量检查

#### 自定义样式

项目使用 UnoCSS 进行样式管理：

```javascript
// uno.config.js
export default defineConfig({
  // 自定义主题色
  theme: {
    colors: {
      primary: {
        50: '#eff6ff',
        // ... 其他颜色
      }
    }
  },
  // 自定义快捷类
  shortcuts: {
    'btn': 'inline-flex items-center justify-center px-4 py-2 border border-transparent text-sm font-medium rounded-md',
    'btn-primary': 'btn text-white bg-primary-600 hover:bg-primary-700'
  }
})
```

#### 集成新的 Markdown 处理器

```javascript
// src/utils/markdown.js
import { marked } from 'marked'

// 自定义 marked 配置
marked.setOptions({
  breaks: true,
  gfm: true,
  // ... 其他配置
})

// 自定义解析器
export function customMarkdownToHtml(markdown) {
  // 自定义处理逻辑
  return marked.parse(markdown)
}
```

### 常见问题

#### Q: 工具加载缓慢怎么办？

A: 检查网络连接，确保使用现代浏览器。考虑使用 CDN 加速静态资源。

#### Q: PDF 导出失败怎么办？

A: 确保内容不为空，检查浏览器是否支持 PDF 生成。某些浏览器可能需要允许弹出窗口。

#### Q: 如何自定义工具样式？

A: 修改 `uno.config.js` 中的主题配置，或在组件中添加自定义样式类。

#### Q: 支持哪些 Markdown 扩展语法？

A: 支持 GitHub Flavored Markdown (GFM) 的所有语法，包括表格、任务列表、删除线等。
